## Tại sao sử dụng MDX?

MDX cho phép chúng ta kết hợp Markdown với React components, giúp tạo ra các bài viết tương tác và sinh động.

### Ví dụ code

```python
# Ví dụ về một mô hình ML đơn giản
import numpy as np
from sklearn.linear_model import LinearRegression

# Dữ liệu mẫu
X = np.array([[1], [2], [3], [4], [5]])
y = np.array([2, 4, 6, 8, 10])

# Training model
model = LinearRegression()
model.fit(X, y)

# Prediction
print(f"Hệ số: {model.coef_[0]}")
print(f"Intercept: {model.intercept_}")
```
